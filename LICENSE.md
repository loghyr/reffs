This project is compliant with the [REUSE](https://reuse.software/spec/)
specification, each file contains [SPDX](https://spdx.org/specifications)
tags, the full licenses can be found in the `LICENSES/` directory.

The code is distributed under `GPL-2.0+`. See `LICENSES/GPL-2.0` for
license details. Refer to the individual file headers for details.

derived from IETF Trust materials. These files are made available under
the BSD-3-Clause license, as described in the IETF Trust Legal Provisions:
